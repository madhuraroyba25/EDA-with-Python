# EDA with Python

## GitHub Repository
https://github.com/madhuraroyba25/EDA-with-Python

## Description
This project includes exploratory data analysis (EDA) on housing data using Python libraries such as pandas, seaborn, and matplotlib.

## Analysis includes:
- Scatter plot (Price vs Living Area)
- Price distribution histogram (waterfront vs non-waterfront)
- Time series analysis of monthly home sales
- Boxplot of price distribution by grade
